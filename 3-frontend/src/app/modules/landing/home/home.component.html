No, el sistema NO incluye facturación electrónica, ya que está diseñado para ser utilizado por clientes de diferentes países, y cada uno tiene sus propias normativas fiscales.

El Sistema está desarrollado con SQL, .NET 9 y Angular 18, siguiendo una arquitectura limpia (DDD). Incluye autenticación con JWT, validaciones con Fluent Validation, generación de reportes en Excel y PDF, y almacenamiento de fotos y archivos en la nube con Cloudinary.

El sistema utiliza **Entity Framework Core** para interactuar con la base de datos SQL Server. Se gestionan migraciones automáticas, relaciones entre entidades definidas, y consultas optimizadas para asegurar el rendimiento. Además, se emplean transacciones para garantizar la integridad de los datos, asegurando que cada cambio en la base de datos se realice de forma segura y consistente.

El Backend incluye servicios de email, recuperación de contraseña, encriptación segura, generación de logs, manejo de excepciones globales, API REST con Swagger y soporte para múltiples zonas horarias.

El Frontend desarrollado con Angular 18 es una PWA. Incluye gráficos dinámicos con Apex Charts, validaciones reactivas, carga dinámica con Skeleton, subida de fotos, notificaciones y modales dinámicos para una mejor experiencia de usuario.

El Sistema incluye módulos para Seguridad, Usuarios, Clientes, Inventario y Ventas. Cada módulo permite gestión completa, filtros avanzados, reportes detallados y opciones para habilitar o deshabilitar registros.
